#### Dva.js的使用

dva 是dva 是 react 和 redux 的最佳实践，最核心的是提供了 app.model 

----------

dva初始化项目

1. 安装 dva-cli
npm install dva-cli -g

2. 创建项目：Dva-test项目名
dva new Dva-demo

3. 进入项目
cd Dva-test

4. 启动项目
npm  start

5. 启动界面
